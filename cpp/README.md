# Raspberry Pi gpiox C++ library

## Libraries:
```libgpiox_arm32.a``` for 32bit OS<br>
```libgpiox_arm64.a``` for 64bit OS<br>

## Header File:
```gpiox_lib.h```<br>
namespace: ```gpiox``` 

## Build:
```
make
```
## Run:
```
./<filename>
```

For more info see gpiox_lib.h and API.md
